# Personal Portfolio

## Overview
This is a personal portfolio website built to showcase my projects, skills, and experience. The site is fully responsive and uses **HTML, CSS, and JavaScript** to dynamically display content from JSON files. It also includes client-side form validation and a project spotlight section.

## Features
- **About Me Section**: Displays a personal introduction and headshot from `aboutMeData.json`.
- **Projects Section**: 
  - Dynamically generates project cards from `projectsData.json`.
  - Clicking a project card updates the project spotlight.
  - Responsive layout: horizontal scrolling on mobile, vertical on desktop.
- **Form Validation**:
  - Validates email format and message length.
  - Live character counter for the message field.
  - Displays error messages for invalid input.

## Tech Stack
- HTML5 & CSS3 (responsive design)
- JavaScript (DOM manipulation & event handling)
- JSON (for project and about me data)
